# ScholarlyForge AI

## Overview
ScholarlyForge AI is an advanced Streamlit application engineered to automate the architectural design of scientific papers. By leveraging the Google Gemini API, Semantic Scholar, and arXiv, the system processes user research concepts, experimental source code, and custom bibliographies to synthesize highly structured outlines, functional LaTeX templates, and BibTeX citations.
